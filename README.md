# Методы сравнения изображения

## Перцептивный хэш - phash

### Реализация в Python

Установка дополнительных библиотек

```sh
pip3 install Pillow
pip3 install ImageHash
```

Как выглядит хэш:
```sh
98ace3532c6d6c39
```
